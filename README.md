## Install the dependancies

```bash
$ npm i
```

## Run a local server with hot reloading

```bash
npm run devserver
```

## Local development build

```bash
npm run build
```

Output is in `build/`

## Production build

```bash
NODE_ENV=production npm run build
```

Output is in `build-production/`

## Deployment

To deploy on Github Pages:

```bash
npm run deploy
```
